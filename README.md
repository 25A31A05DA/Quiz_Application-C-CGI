# Web-Based Quiz Application (C + CGI)

This project is a web-based quiz application implemented using C programming via CGI.
---
## Versions
- v1: Terminal-based quiz application (C)
- v2: Web-based version using HTML, CSS, Apache, and CGI
---
## Tech Stack
- Frontend: HTML, CSS
- Backend: C (CGI)
- Server: Apache (XAMPP)
---
## Description
The application accepts user input through a web form, processes answers on the server using a C program, and dynamically generates a result page with scores.
---
## How it Works
1. User opens the quiz page in a browser
2. HTML form submits data via HTTP GET
3. Apache executes the C program using CGI
4. C backend evaluates answers and returns HTML output
---
